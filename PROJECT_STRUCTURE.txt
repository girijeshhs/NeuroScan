🚀 QUICK START:

   1. Start everything:
      ./start_all.sh

   2. Or manually:
      Terminal 1: cd backend && python3 app.py
      Terminal 2: cd frontend && npm run dev

   3. Access:
      Backend:  http://127.0.0.1:5000
      Frontend: http://localhost:5173
